SELECTED WORK

HX5

The HX5 headphones feature LED lights that synchronize with the rhythm of the music.

This particular model is crafted using Solid, enabling enhanced precision in measurements and facilitating assembly tests.

MARIA

Inspired by The Veiled Virgin.

This model was made entirely in Blender. Its main inspiration was the use of fabric/cloth physics.

DRAGON

This dragon is based on an illustration I made. Turning it into a 3D object was the result of my quest to gain more experience in the field of sculpting and rendering.

Both the 3D model and the shading and texturing are completely done in Blender.

TIGER

The TIGER sports shoes offer exceptional comfort and superior traction.

This particular model is crafted using Rhino, providing extensive flexibility in the modeling process, allowing for greater creative freedom without strict adherence to precise measurements.

BOCCHI REX

Bocchi is a character from a manga with an anime adaptation (Bocchi the Rock).

This particular model is crafted using Blender and textures created with Substance 3D Painter, providing extensive flexibility in the modeling process, allowing for greater creative freedom and create artistic models.
